The JQuery Inline Form

This is a small JQuery plugin that turns any element into a form that toggles intelligently. For CRUD applications where people need to be able to update data, this works pretty well

Requires Bootstrap 3.0.x

This plugin uses Twitter Bootstrap for styling and icons so your page will have to include Bootstrap js and css in order to work
